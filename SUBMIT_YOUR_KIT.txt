WaterkoofAI Airdrop Script Kit — Submission Guide
==================================================
t.me/WaterkoofAI_Bot


WHAT GETS SUBMITTED AND REVIEWED
----------------------------------
You submit a complete Kit for a specific Web3 project.
If it passes review, WaterkoofAI will publish it to the official GitHub.
Users can then find, download, and run it.

Your Kit must contain exactly these four files:

  STEP1_Drop_me_to_any_AI.md
  STEP2_Fill_your_wallet_info.json
  project_auto.py                   (rename to your project, e.g. arbitrum_auto.py)
  If_you_dont_use_AI_read_me.txt


HOW TO SUBMIT
--------------
1. Create a public GitHub repository for your Kit.
   Name it clearly, e.g.: waterkoofai-arbitrum-kit

2. Upload all four files to the repository root.

3. Make sure the repository is set to PUBLIC.

4. Open Telegram and go to: t.me/WaterkoofAI_Bot

5. Send the following message to the bot:

   [KIT SUBMISSION]
   Project: [Project Name]
   GitHub: [your repository URL]
   Coverage: [X / X tasks, ~X% automated]
   Tested on: [Windows / macOS / Linux]
   Notes: [anything you want to mention]

6. Wait for a response. Reviews typically take 1-3 days.


WHAT HAPPENS DURING REVIEW
-----------------------------
WaterkoofAI will check the following:

SECURITY CHECK (must pass — any failure = instant rejection):
  - No private key or seed phrase fields anywhere in the script
  - No external data transmission (no POST requests to unknown URLs)
  - No auto-approval of MetaMask transactions
  - Script is fully readable and auditable

FUNCTIONALITY CHECK:
  - Script runs without crashing on at least one OS
  - Coverage declaration matches actual behavior
  - ACTION NEEDED prompts are clear and accurate
  - Placeholder wallets are filtered out correctly

DOCUMENTATION CHECK:
  - All four files present with correct names
  - STEP1 covers Windows, macOS, and Linux commands
  - If_you_dont_use_AI_read_me.txt is complete and accurate
  - Coverage declaration present in both script header and STEP1

QUALITY CHECK:
  - No misleading claims about points or airdrop amounts
  - Script handles errors gracefully without crashing
  - Works for multiple wallets


REVIEW OUTCOMES
----------------
APPROVED:
  Your Kit will be added to the WaterkoofAI GitHub under the official collection.
  You will receive an initial rating score (see RATING_RUBRIC.txt).
  Your Telegram username will be credited as the author.

REVISION REQUESTED:
  You will receive specific feedback on what needs to be fixed.
  Resubmit after making the changes.

REJECTED:
  Kits that fail the security check are permanently rejected.
  Other rejections include: script does not run, coverage is fabricated,
  documentation is missing or misleading.


AFTER APPROVAL
---------------
Your Kit will be listed in the WaterkoofAI GitHub with:
  - Project name
  - Your credited username
  - Coverage rate
  - Initial rating score
  - Link to your repository

Users can download directly from your repository link.
If you update your Kit, notify the bot so the rating can be re-evaluated.


RULES AND RESPONSIBILITIES
---------------------------
- You are responsible for keeping your Kit accurate as projects update
- Do not submit Kits for projects you have not personally tested
- Do not copy another developer's Kit and resubmit it as your own
- WaterkoofAI reserves the right to remove any Kit at any time
- Submitting a Kit does not guarantee approval or listing
